# HawkeyeMonitor 实时监控平台

能做什么？

    资产发现:
        ·对目标范围进行探测,发现其中存活的端口及资产,梳理资产分布情况

    目标扫描:
        ·对发现资产进行安全检测,根据漏洞评估资产风险状况,保障资产安全性

    资产管理:
        ·实施一键式资产发现并分组管理

亮点优势？

    资产检测保障安全
        ·对发现资产进行安全检测, 系统提供IP资产检测, WEB资产检测以及弱口令检测功能

    内容监控
        ·自定义时间周期,对管理资产进行安全动态监测,包括暗链,篡改,敏感词语监控

    态势分析
        ·从漏洞,指纹,事件,资产等几个维度对发现资产进行分析可视化展示.

面对群体?

    企业
      ·针对中小型企业内部资产管理
